Class: JsonTransformer

JsonTransformer

A class to recursivley transform JSON values by applying JSON transformers.

new JsonTransformer(_ [, init] [, transformer])

Parameters:
Name Type Argument Default Description
_ JsonTransformerParameters An object containing the following attributes
init any <optional>
 An object that may be used to initialize the transformer. Used by subclasses.
_.data Data <optional>
 {} A data object that is passed as environment to the transformers. It can be used by transformers (defined via subclassing) to replace or compute certain JSON valuess.
_.level number <optional>
 0 The current level of the JSON value. The level of the top JasonValue (usually) is equal to 0. The level of its children is 1, the level of the grand children 2, etc.
transformer JsonTransformer <optional>
 A transformer to which the JSON value is piped. This transformer may transform the JSON value before it is piped. Moreover, the result of the pip etransformer may be transforemd further by this transfoer.

JsonValue

A JSON value is either a primitive value (JsonPrimitive: string, number, boolean, or null), or an array of JSON values (JsonArray) or a map object the members of which consist of a key string and a JSON value (JsonMap)

undefined is no regular JSON value. But it is needed to be able to uses optional parameters within the JSON transformers.

Type:
  • JsonPrimitive | JsonArray | JsonMap | undefined

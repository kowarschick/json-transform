(static, constant) JsonFunctionAverage :JsonFunctionDescriptor

This constant defines the average aggregator. It returns the average of the elements of an array of numbers. The average of an empty array is not defined (NaN).

Examples

import { JsonTransformerFunction } from '@wljkowa/json-transformer';
import { JsonFunctionAverage }     from '@wljkowa/json-transformer';

const t = new JsonTransformerFunction({ init: [JsonFunctionAverage] });

t.transform({ value: ["$average", 5, 4] }); // => 4.5
t.transform({ value: ["$average", 5] });    // => 5
t.transform({ value: ["$average"] });       // => NaN

t.transform({ value:
              { "$function": "$average"
                "$value":    [5, 4]
              }
           });                              // => 4.5
t.transform({ value:
              { "$function": "$average"
                "$value":    [5]
              }
           });                              // => 5
t.transform({ value:
              { "$function": "$average"
                "$value":    []
              }
           });                              // => NaN

Methods

(static) aggregate(_, begin) → {JsonValue}

Aggregates an array of JSON values to one JSON value. If the length is equal to zero, _.init.default is returned. Optionally, a initialization function can be stated to compute the default value. Otherwise _.init.aggregate is succesively applied to the aggregation value and the elements of the array. Optionally, a finalization function can be used to further process the result of the aggregation.
